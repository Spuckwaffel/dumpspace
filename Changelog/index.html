Latest Updates

All the latest Updates to the page.

  1. January 2024

    Inheritance Viewer

    This update introduces the Inheritance Viewer, allowing you to visualize a graph of all classes that inherit from a specified target class, with customizable depth settings.

  2. August 2024

    Dumpspace generation 1.02.02 update

    The 1.02.02 Update adds the Array Dimension attribute, so C-Style arrays get shown properly on the website. Besides that, many UI improvements have been made and several bugs have been fixed.

  3. July 2024

    Starboard Update

    A Starboard has been introduced in Dumpspace showcasing the top 5 contributors of the webite!

  4. March 2024

    Full mobile support

    Besides many UI improvements and small improvements on Global Search, mobile support has been finished even for small devices.

  5. February 2024

    Global Search and API launch

    Global Search and the Dumpspace API has been added for all listed Games.

  6. December 2023

    First public release and accouncement

    The project has been released to the public for testing and adding Games.

  7. November 2023

    Final touches before release

    The game viewer has been finished and final UI touches have been made.

  8. September 2023

    First look

    The Dumpspace went public on GitHub and for testing!

  9. August 2023

    Project start

    Start of the Dumpspace project.
